1. Open: C:\xampp\apache\conf\extra\httpd-vhosts.conf

Add lines:
<VirtualHost *:80>
       DocumentRoot "C:/xampp/htdocs/html/JAT/skullframework/_skull/"
       ServerName skull.com
</VirtualHost>
<VirtualHost *:80>
       DocumentRoot "C:/xampp/htdocs/html/JAT/skullframework/cms/"
       ServerName cms.com
</VirtualHost>

2. Right click notepad and select Run as administrator then Ctrl + O to open C:\Windows\System32\drivers\etc\hosts

Add lines:
        127.0.0.1       skull.com
        127.0.0.1       cms.com

3. Restart Apache

4. Open: C:\xampp\htdocs\html\JAT\skullframework\cms\.htaccess

#---------
#Edit v0.0.0 which refers to the build of App you want to use.
#Build Numb
RewriteRule (.*) - [E=build:v0.0.0]

#Use this if you're using different domain for Skull and App.
#Skull Vars
RewriteRule (.*) - [E=skull_root:C:/_/skullframework/_skull/]
RewriteRule (.*) - [E=skull_domain:http://skull.com/]

#App Vars
RewriteRule (.*) - [E=root:%{DOCUMENT_ROOT}%{ENV:build}/]
RewriteRule (.*) - [E=domain:http://%{HTTP_HOST}/]



#Use this if you're using same domain for Skull and App.
#Skull Vars
RewriteRule (.*) - [E=skull_root:%{DOCUMENT_ROOT}skullframework/_skull/]
RewriteRule (.*) - [E=skull_domain:http://%{HTTP_HOST}/skullframework/_skull/]

#App Vars
RewriteRule (.*) - [E=root:%{DOCUMENT_ROOT}skullframework/cms/%{ENV:build}/]
RewriteRule (.*) - [E=domain:http://%{HTTP_HOST}/skullframework/cms/]
#---------

5. Open browser and visit cms.com

6. If has errors check: C:\xampp\htdocs\html\JAT\skullframework\cms\v0.0.0\_spec\func\config.php
