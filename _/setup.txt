1. Open C:\xampp\apache\conf\extra\httpd-vhosts.conf

Add lines:
<VirtualHost *:80>
       DocumentRoot "C:/_/JAT/skullframework/_skull/"
       ServerName skull.com
       <Directory "C:/_/JAT/skullframework/_skull/">
              Options Indexes FollowSymLinks Includes ExecCGI
              AllowOverride All
              Require all granted
       </Directory>
</VirtualHost>
<VirtualHost *:443>
       DocumentRoot "C:/_/JAT/skullframework/_skull/"
       ServerName skull.com
       SSLEngine On
       SSLCertificateFile "C:/xampp/apache/conf/ssl.crt/server.crt"
       SSLCertificateKeyFile "C:/xampp/apache/conf/ssl.key/server.key"
       <Directory "C:/_/JAT/skullframework/_skull/">
              Options Indexes FollowSymLinks Includes ExecCGI
              AllowOverride All
              Require all granted
       </Directory>
</VirtualHost>
<VirtualHost *:80>
       DocumentRoot "C:/_/JAT/skullframework/app/"
       ServerName app.com
       <Directory "C:/_/JAT/skullframework/app/">
              Options Indexes FollowSymLinks Includes ExecCGI
              AllowOverride All
              Require all granted
       </Directory>
</VirtualHost>
<VirtualHost *:443>
       DocumentRoot "C:/_/JAT/skullframework/app/"
       ServerName app.com
       SSLEngine On
       SSLCertificateFile "C:/xampp/apache/conf/ssl.crt/server.crt"
       SSLCertificateKeyFile "C:/xampp/apache/conf/ssl.key/server.key"
       <Directory "C:/_/JAT/skullframework/app/">
              Options Indexes FollowSymLinks Includes ExecCGI
              AllowOverride All
              Require all granted
       </Directory>
</VirtualHost>

Important:
Never forget "AllowOverride All" otherwise some declarations in .htaccess will cause the containing folder to be hidden in localhost and accessing the hidden folder will result to "Internal Server Error".

Error Declarations in .htaccess:
DirectoryIndex jtan.php
php_value post_max_size 300M
php_value upload_max_filesize 300M

2. Right click notepad and select Run as administrator then Ctrl + O to open C:\Windows\System32\drivers\etc\hosts

Add lines:
127.0.0.1       skull.com
127.0.0.1       app.com

3. Restart Apache

4. Open skullframework\app\.htaccess

#---------
#Edit v0.0.0 which refers to the build of App you want to use.
#Build Numb
RewriteRule (.*) - [E=build:v0.0.0]

#Use this if you're using different domain for Skull and App.
#Skull Vars
RewriteRule (.*) - [E=skull_root:C:/_/skullframework/_skull/]
RewriteRule (.*) - [E=skull_domain://skull.com/]

#App Vars
RewriteRule (.*) - [E=root:%{DOCUMENT_ROOT}%{ENV:build}/]
RewriteRule (.*) - [E=domain://%{HTTP_HOST}/]



#Use this if you're using same domain for Skull and App.
#Skull Vars
RewriteRule (.*) - [E=skull_root:%{DOCUMENT_ROOT}skullframework/_skull/]
RewriteRule (.*) - [E=skull_domain://%{HTTP_HOST}/skullframework/_skull/]

#App Vars
RewriteRule (.*) - [E=root:%{DOCUMENT_ROOT}skullframework/app/%{ENV:build}/]
RewriteRule (.*) - [E=domain://%{HTTP_HOST}/skullframework/app/]
#---------

5. Open browser and visit http://app.com or https://app.com. These are both working because of enabling VirtualHost *:80 and VirtualHost *:443

6. If has errors check skullframework\app\v0.0.0\_spec\func\config.php
